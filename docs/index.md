# ComfyCollectorNodes

A collection of custom nodes for ComfyUI covering conditioning, latents, LoRA
loading, prompt construction, and workflow utilities.

!!! warning "Work in progress"
    This pack and its documentation are under active development. Node behavior
    and these docs may change without notice.

## Where to start

- **[Getting Started](getting-started.md)** — install the pack and load your first node.
- **[Nodes](nodes/index.md)** — reference for every node, grouped by category.
- **[Concepts](concepts/index.md)** — how the pieces fit together: the conditioning
  pipeline, WAN temporal conditioning, and more.
